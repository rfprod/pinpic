User Stories
An unauthenticated user can login with Twitter or sign up / login with email and password. An authenticated user can link to images. An authenticated user can delete images that the user has linked to. An authenticated user can see a Pinterest-style wall of all the images the user has linked to. An unauthenticated user can browse other users' walls of images. An authenticated user, if the user uploads an image that is broken, it will be replaced by a placeholder image.
